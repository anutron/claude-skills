## Skills Available

All skills live in **this project** at `.claude/skills/` — never in global `~/.claude/skills/`.

**Naming Convention:**
- `personal-` prefix — personal skills (calendar, daily rhythm, expenses). Excluded from publishing automatically.
- `work-` prefix — work-specific skills. Also excluded from publishing.
- No prefix — generic/publishable skills (dev, test, guard, etc.)
